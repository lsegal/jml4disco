Introduction

This section provides preliminary prover benchmarking results. The goal is to determine the average performance of each prover that is used in JML4 including Simplify, CVC3 and Isabelle.

Benchmark Environment

SOEN 490 H-837 Lab Computer
Processor Intel Core 2 Duo E6300 1.86GHz
FSB 1066 MHz
L2 Cache 2MB
Memory 2GB, 667 MHz
OS Fedora 9
IDE Eclipse Ganymede

Benchmark Settings

Benchmark Tool Eclipse TPTP, Manual Code, Eclipse Memory Analyzer
Provers Simplify, CVC3, Negation Simplify, Isabelle
Benchmark Method Execution times (target only vc proving methods), Memory Allocation of VCs
Filter Profile only classes under org.jmlspecs.jml4.esc.provercoordinator.prover.* package
VCs Caching Clear before profiling
