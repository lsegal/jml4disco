Java Coverage Traceability

1. Introduction

This document describes the coverage of the Boogie Disco component in terms of Java functionality. Each Java function point is traced to an implementation that translates the Java to Boogie. Each Java Node (named with the JN prefix followed by a unique number) can be found in the BoogieTraceability document, which maps the Java node implementations to their respective tests.

The legend for this matrix can be found below.
