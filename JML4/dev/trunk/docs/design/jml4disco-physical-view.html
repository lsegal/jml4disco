Physical View (Deployment View)

1. Introduction

This wiki page describes the physical (deployment) view of the JML4Disco distributed subsystem. Each Node represents a type of physical computer / terminal and each square inside it represents an important element that originates from the entity that surrounds it. The entities are connected by a network protocols (e.g. HTTP).
