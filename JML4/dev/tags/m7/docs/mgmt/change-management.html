During Milestone 6, a complete diff was done between JML4 code (downloaded from sourceforge on the 6th of February) and the source code for the JML4 Disco project, revision 419, found in SVN on the 23rd of February 2009. The purpose of this document is to list the differences between the two code bases. This document is meant to be used in conjunction with the Impact Analysis document. This document lists changes in a more granular, low-level way while the Impact Analysis lists the high level purpose and justification for all changes.

  • Implements java.io.Serializable to allow serialization
  • Instance variable of type TypeBinding is large and not completely necessary to include in a serialized VC object for proving.
    Given transient specification. Custom serialization methods were written to handle the necessary components of this objects.
    • Custom Serialization: added private methods serializeTypeBinding(...) and deserializeTypeBinding(...) used by custom serialization overridden methodswriteObject(...) & readObject(...).
      • Method serializeTypeBinding(...) accepts a parameter of type TypeBinding and returns an easily-serialized string which can be converted back to an instance of TypeBinding.
      • Method serializeTypeBinding(...) is the inverse of serializeTypeBinding(...); it accepts a parameter of type string and returns an instance of TypeBinding.

1.3 Added Packages/Classes

Package org.jmlspecs.jml4.boogie

I.A. 3.2
  • Boogie: A compiler extension that enables Java code to be passed through Microsoft Boogie in order to formally verify JML DbC annotations.
  • BoogieAdapter: Wraps the Boogie runtime into a Java class that can be called from Eclipse's compilation stack.
  • BoogieSource: Represents a buffer of Boogie source code that will be passed to a Boogie runtime object.
  • BoogieSourcePoint: Represents a point in a Boogie source code file. Used to map points in Boogie source back to a Java source file.
  • BoogieVisitor: Visitor for the boogie re-ordered AST

Package org.jmlspecs.jml4.boogie.ast

I.A. 3.2
  • New package holding all Boogie specific AST Nodes.
    • AssertStatement, Assignment, AssumeStatement, Axiom, BinaryExpression, BoogieNode, BooleanLiteral, BreakStatement, CallStatement, Comment, ConstStatement, DecoratorVisitor, EmptyStatement, Expression, FunctionCall, FunctionDeclaration, IfStatement, IntLiteral, LabelStatement, MapTypeReference, MapVariableReference, Procedure, Program, RemoveLocal, ResultReference, ReturnStatement, Scope, Statement, TokenLiteral, TypeDeclaration, TypeDeclaratorVisitor, TypeReference, UnaryExpression, VariableDeclaration, VariableLengthReference, VariableReference, VariableStatement, Visitor, WhileStatement
