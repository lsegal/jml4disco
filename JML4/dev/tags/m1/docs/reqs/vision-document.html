Vision Document

1. Introduction

The Vision document defines our team's vision for the JML4 Project. In the project we will aid the DSRG team in developing solutions intended for JAVA developers using the Java Modeling Language (JML) to design by contract (DBC). We will contribute to the project by working on 3 features to further improve JML4:

  • Distributed Discharging of VCs
  • Integration of Boogie
  • JMLUnit component for JML4

This document will present these three features as products with corresponding problem statements of their own.
