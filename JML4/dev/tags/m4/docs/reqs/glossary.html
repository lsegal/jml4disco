Glossary

ADT Abstract Data Type
AST Abstract syntax tree
BISL Behavior Interface Specification Language
Boogie Static program verifier for Spec# by Micorsoft
BoogiePL Intermediate language for program analysis and program verification, used as input to Boogie
CIL Common Intermediate Language, Microsoft’s equivalent of Java bytecode
CORBA Common Object Request Broker Architecture
DSRG Dependable Software Research Group
ESC Extended Static Checking, a limited, fully automated form of static checking
ESC4 Extended Static Checking component of JML4
ESC4Web Initial version of servlets used in distributed proving using ESC4
FPV Full Program Verification, a complete, interactive form of static checking
GC Guarded Command, a statement in a simple verification language by Dijkstra
IDE Integrated Development Environment
IVE Integrated development and Verification Environment
JDT Java Development Toolkit, provides the Java environment in Eclipse
JML Java Modeling Language, a BISL for Java
JML4 Next Generation Integrated Verification Environment
JMLUnit JML’s Unit Testing Tool
OO Object Oriented
Pretty PrintingError highlighting in java code, red squiggly underlined code
PV Program Verification
RAC Runtime Assertion Checking, a form of dynamic verification.
RMI Remote Method Invocation
SVN Subversion, program control system, used to maintain current and historical data on files
TPTP Eclipse Test & Performance Tools Platform Profiling tool can be used to profile Eclipse plug-ins
WP Weakest Precondition
VCSE Verification-Centric Software Engineering
VC Verification Condition, a formula to be proved to ensure a property holds of given code.
VCProgramCollection of VCs
VSI Verified Software Initiative
